# IssueTrackerApp Dev Blog

Welcome to the development blog for [IssueTrackerApp](https://github.com/mpaulosky/IssueTrackerApp), a modern issue tracking application built with .NET Aspire, Blazor, and MongoDB.

Here we document new features, architectural decisions, testing milestones, and the story of how this project evolves. Whether you're a contributor, developer, or just curious about .NET best practices, you'll find posts covering the technical decisions and work that power IssueTrackerApp.
