# Web Layer

This project contains the Blazor Server UI with Interactive Server Rendering.

## Folder Structure

- **Components/Pages/** - Blazor page components
- **Components/Layout/** - Layout components (MainLayout, NavMenu, etc.)
- **Features/** - Feature-specific UI components organized by vertical slice

## Configuration

- Interactive Server rendering mode
- Antiforgery tokens enabled
- Static asset mapping
- Error handling and status code pages
